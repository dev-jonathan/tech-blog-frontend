# Tech Blog

Um blog sobre tecnologia construído com React, TypeScript e Tailwind CSS.

## Stack

- **React 19** - Biblioteca UI
- **TypeScript** - Type safety
- **Vite (Rolldown)** - Build tool
- **Tailwind CSS v4** - Estilização
- **shadcn/ui** - Componentes UI
- **Lucide React** - Ícones

## Instalação

```bash
# Instalar dependências
pnpm install

# Rodar em desenvolvimento
pnpm dev

# Build para produção
pnpm build
```
